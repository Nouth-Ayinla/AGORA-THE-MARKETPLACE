# Agora

Nigeria's first marketplace built for student Entrepreneurs. Sell products and services, keep 90% of your earnings, and get paid in 3 days.

## Project Structure

```
├── index.html   # Main HTML structure
├── styles.css   # Custom CSS (animations, utilities)
├── script.js    # JavaScript (dark mode, scroll effects, forms)
└── README.md
```

## Features

- Sell products **and** services from one profile
- 10% commission (vs 20% on competitors)
- 3-day payment turnaround
- Exam Mode — pause your shop during exam week
- No CAC registration required, just a student ID
- 24/7 human support

## Getting Started

No build tools needed. Just open `index.html` in a browser.

The project uses:

- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Google Fonts — Outfit](https://fonts.google.com/specimen/Outfit)
- [Material Icons](https://fonts.google.com/icons)
